# Capitals Dataset:
### Overview:

This dataset contains information about various aspects of capitals, focusing on text generation and sentiment analysis. The data were collected from multiple social media platforms and stored in CSV format. The dataset provides unique insights into the sentiment classification of capital-related sentences.

# Data Source Location and Platforms
### Data Source Location:

The data were collected from various online platforms and stored in CSV files. These CSV files were then uploaded to a GitHub repository for easy access and management. The data is securely maintained within the repository, ensuring confidentiality and integrity.

# Dataset Description
The dataset files are in CSV (Comma Separated Values) format. There are 10 columns in the dataset, each providing unique insights into various aspects of capitals. Below are the types of data for each column along with their descriptions:

# Columns
### Capitals Text Generation

### Type: Text
### Data Format: Raw
### Description: 
Contains text data related to the generation of capital-related sentences.
Sentiment Label

### Type: Categorical
###  Data Format: Analyzed
### Description:
Indicates the sentiment classification of the text (e.g., positive, negative).

# Predicted Capital

### Type: Text
### Data Format: Analyzed
### Description:
Contains the predicted capital names or related information.
### User-Id

### Type: Numeric
### Data Format: Raw
### Description:
Unique identifier for each user.
### Social-Accounts

### Type: Text
### Data Format: Raw
### Description: 
Contains information about users' social media accounts.
# Date

### Type: Date
### Data Format: Raw
### Description: 
Date on which the data was recorded.
# Time

### Type: Time
### Data Format: Raw
### Description:
Time at which the data was recorded.
# Day

### Type: Categorical
### Data Format: Raw
### Description:
Day of the week when the data was recorded.
# Country

### Type: Text
### Data Format: Raw
### Description:
### Name of the country related to the capital data.

# Age-Of-User

### Type: Numeric
### Data Format: Raw
### Description:
### Age of the user providing the data.
# Data Acquisition

The data were acquired using a combination of web scraping and manual curation. Web scraping was primarily done using tools like Selenium and BeautifulSoup to collect data from various social media platforms. Manual curation was employed to ensure data quality and relevance.

# Ethical Considerations
The data collection process adhered to ethical guidelines to ensure the privacy and consent of the users from whom data were collected. All user data were anonymized, and platform data redistribution policies were complied with.
